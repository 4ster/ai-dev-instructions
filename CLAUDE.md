# Проект: Серия инструкций по AI-разработке

## О проекте

Серия md-файлов с инструкциями по разработке с помощью AI-агентов. Каждая инструкция — отдельный md-файл, где команды и шаги для разных ОС скрыты в `<details>`.

## Целевая аудитория

- **Уровень:** Начинающие разработчики
- **ОС:** Windows, macOS, Linux (все три)
- **Язык:** Русский

## Формат инструкций

- Основной текст — общий для всех ОС
- Специфичные команды для ОС — внутри `<details><summary>OS Name</summary>...</details>`
- Таблицы для сравнений и списков инструментов
- Чек-листы для проверки выполнения

## Технические решения

- **Node.js и Claude Desktop:** В Claude Desktop есть настройка "Use built-in Node.js for MCP", которая работает только для Desktop. Для Claude Code CLI Node.js обязателен.
- **MCP серверы в первой инструкции:** Только базовые три (filesystem, github, memory), остальные — ссылки.

## Стиль написания

- Объяснять базовые концепции (что такое CLI, переменные окружения)
- Не перегружать терминологией
- Давать практические примеры
- Указывать "зачем" для каждого шага
