# Инструкции по AI-разработке

Серия пошаговых инструкций по разработке с AI-агентами (Claude, Codex) для начинающих разработчиков.

## Для кого

- Начинающие разработчики, которые хотят использовать AI в своей работе
- Пользователи Windows, macOS и Linux
- Все инструкции на русском языке

## Особенности

- Пошаговые инструкции с объяснением каждого действия
- Команды для всех операционных систем
- Чек-листы для проверки выполнения
- Практические примеры использования

## Инструкции

| № | Инструкция | Описание |
|---|------------|----------|
| 01 | [Установка и настройка среды](01-setup-environment.md) | VS Code, CLI-инструменты, API-ключи, базовые MCP серверы |
| 02 | [Работа с GitHub через AI-агентов](02-github-ai-agents.md) | Git, GitHub CLI, MCP сервер GitHub, практические сценарии |
| 03 | [Интеграция с Firebase](03-firebase-integration.md) | Firebase CLI, Firestore, Authentication, Hosting |
| 04 | [Продвинутые MCP серверы](04-advanced-mcp.md) | fetch, puppeteer, postgres, sqlite, brave-search, telegram |
| 05 | [Настройка WSL для Windows](05-wsl-setup.md) *(опционально)* | WSL2, интеграция с VS Code, Docker, Claude Code в Linux |

## С чего начать

1. Начните с [первой инструкции](01-setup-environment.md) — она содержит базовую настройку среды
2. Далее выберите инструкции в зависимости от ваших задач:
   - Работаете с GitHub → [Инструкция 02](02-github-ai-agents.md)
   - Разрабатываете веб-приложение с бэкендом → [Инструкция 03](03-firebase-integration.md)
   - Нужны дополнительные возможности AI → [Инструкция 04](04-advanced-mcp.md)
   - *(Опционально)* Используете Windows и хотите Linux-среду → [Инструкция 05](05-wsl-setup.md)

## Что вы получите

После прохождения инструкций у вас будет настроенная среда, в которой AI-агенты смогут:

- Писать и редактировать код
- Работать с Git и GitHub (коммиты, PR, issues)
- Взаимодействовать с Firebase (база данных, хостинг, авторизация)
- Делать HTTP-запросы и парсить веб-страницы
- Работать с базами данных (PostgreSQL, SQLite)
- Выполнять команды в терминале
