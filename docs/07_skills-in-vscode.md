## 1. Что добавилось в VS Code 1.108

В версии **Visual Studio Code 1.108** появилась экспериментальная поддержка **Agent Skills** — механизма расширения возможностей AI‑агентов (например, GitHub Copilot) за счёт проектно‑специфичных навыков.

Agent Skills позволяют:
- описывать знания и правила, специфичные для проекта;
- подключать инструкции, примеры и сценарии;
- автоматически подгружать их в контекст AI‑чата только при необходимости.

Дополнительно в обновлении:
- улучшена работа AI‑чата и управление сессиями;
- повышена стабильность редактора и терминала;
- улучшена интеграция с Git и расширениями.

## 2. Где хранить skills и что такое `SKILL.md`

### Рекомендуемая структура проекта

```text
project-root/
├── .github/
│   └── skills/
│       ├── skill-name-1/
│       │   └── SKILL.md
│       ├── skill-name-2/
│       │   └── SKILL.md
│       └── README.md
├── src/
├── .vscode/
└── README.md
```

По умолчанию VS Code ищет skills в следующих местах:
- `.github/skills/` — основной путь
- `.claude/skills/` — альтернативный путь (для совместимости)

Каждая подпапка внутри `skills` — это отдельный навык (skill).

---

### Файл `SKILL.md`

`SKILL.md` — это декларативное описание навыка, которое использует AI‑агент.

В нём описываются:
- назначение навыка;
- условия, при которых он должен применяться;
- инструкции для агента;
- примеры поведения.

VS Code автоматически анализирует `SKILL.md` и подгружает его **только тогда**, когда запрос пользователя соответствует контексту навыка.

## 3. Что теперь умеет VS Code с Agent Skills

С поддержкой skills VS Code получил возможность:

### Контекстно‑зависимое обучение AI
AI‑агент понимает правила и соглашения конкретного проекта, не смешивая их с глобальными инструкциями.

### Автоматизация рутинных задач
Skills можно использовать для:
- рефакторинга по внутренним стандартам;
- генерации типового кода;
- проверки структуры проекта;
- применения архитектурных правил.

### Масштабируемость
- skills можно версионировать вместе с кодом;
- можно хранить несколько навыков для разных ролей (backend, devops, docs);
- навыки не мешают друг другу и подключаются по требованию.

## 4. Пример Agent Skill

### Структура

```text
.github/
└── skills/
    └── js-formatting/
        └── SKILL.md
```

### Пример `SKILL.md`

```markdown
# name: js-formatting
# description: Форматирование JavaScript-кода по стандартам проекта

## when
Use this skill when the user asks to format, refactor, or standardize JavaScript code.

## instructions
- Приводи код к читаемому виду
- Используй единый стиль отступов
- Избегай неявных преобразований типов
- Следуй принципу читаемости прежде компактности

## examples
Input:
  format this JS snippet

Output:
  Code formatted according to project rules
```

## Как включить Agent Skills в VS Code

1. Открой настройки VS Code
2. Найди параметр `chat.useAgentSkills`
3. Установи значение `true`
4. Перезапусти VS Code

После этого AI‑чат сможет автоматически использовать skills из проекта.
